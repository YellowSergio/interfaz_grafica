Descripción
Esta aplicación permite a los usuarios gestionar una lista de tareas pendientes. Los usuarios pueden añadir tareas, eliminarlas y marcarlas como completadas. Cada vez que se realiza una acción, la lista se actualiza y se refleja en una interfaz gráfica de usuario (GUI) creada con Tkinter.

Características:
Agregar tarea: Permite añadir nuevas tareas a la lista.
Eliminar tarea: Elimina una tarea seleccionada de la lista.
Marcar como completada: Marca una tarea como completada, añadiendo el texto "(Hecha)".
Actualización del estado: Muestra el número total de tareas pendientes en una etiqueta.
Requisitos
Python 3.x
Librería Tkinter (incluida en la instalación estándar de Python)
Instalación
Clona este repositorio o descarga los archivos.
Asegúrate de tener Python 3 instalado en tu sistema.
Uso
Ejecuta el script task_manager.py en tu terminal o entorno de desarrollo de Python.
Aparecerá una ventana donde podrás gestionar tus tareas:
Escribe la tarea en el campo de entrada y presiona "Agregar Tarea" para añadirla.
Selecciona una tarea de la lista para eliminarla o marcarla como completada.
